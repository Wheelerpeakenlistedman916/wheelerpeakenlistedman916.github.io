📋 About the Software

ovstage provides a core foundation for complex simulations. It helps software handle large amounts of data for physics, 3D rendering, and robot animation. The library connects different tools that use the OpenUSD format. It moves data quickly by using your computer graphics card. This makes simulations smoother and faster for users working with digital environments.

Once you install the software, you can initialize it through your terminal or compatible simulation environment. The library works in the background to speed up your data tasks. You do not need to configure complex settings for basic use.

🛡️ Support and Troubleshooting

If you encounter issues during installation, check your graphics card drivers first. Most errors occur because the system cannot talk to the GPU correctly. Ensure you use the latest drivers from the manufacturer. If the program fails to start, verify that you have sufficient disk space and that your user account has administrative permissions.

The library creates small log files during operation. You can find these files in the installation folder under the "logs" directory. Review these files if you need help from technical support.
